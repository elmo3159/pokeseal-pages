利用規約

スナドロ(Snap Drop)

制定日：2026年6月12日

この利用規約（以下「本規約」）は、合同会社 Dream Noark（以下「当社」）が提供するアプリ「スナドロ(Snap Drop)」および関連するWebページ（以下あわせて「本サービス」）の利用条件を定めるものです。利用者は、本サービスを利用することで本規約に同意したものとみなされます。

第1条（定義）

  1. 運営ユーザー：本サービスの運営モードを利用し、撮影・QRコード発行等を行う利用者（イベント運営者・出演者等）をいいます。
  2. ファンユーザー：本サービスのファンモードまたはWebページを利用し、スナドロを受け取り・コレクションする利用者をいいます。
  3. スナドロ：本サービスで撮影・生成された写真・動画・落書きタイムラプス・音声メッセージ等のデジタルコンテンツをいいます。
  4. トレカ：運営ユーザーが設定し、スナドロ受け取り時に抽選で提供されるデジタルカードをいいます。

第2条（アカウント）

  1. 本サービスは登録なし（匿名アカウント）で利用を開始できます。購入の保存・複数端末での利用・データの復元には、メールアドレスとパスワードによるアカウント登録が必要です。
  2. 運営ユーザーは、1つのアカウントをグループ（店舗・チーム等）内の複数端末で共有して利用できます。共有するパスワードの管理責任は運営ユーザーが負います。
  3. アカウントを登録せずアプリを削除した場合、匿名アカウントのデータ（トレカ・残高等）には二度とアクセスできなくなります。端末内にのみ保存されたスナドロは、アプリの削除とともに失われます。
  4. 不正アクセス・なりすまし等が判明した場合、当社はアカウントを停止することがあります。

第3条（課金）

  1. 本サービスの課金はすべて App Store / Google Play のアプリ内課金として処理されます。
  2. 撮影チケット（消耗型）：購入したチケットは当社サーバーで管理されます。チケットはストアの「購入の復元」では復元できないため、購入にはアカウント登録が必要です。使用済みチケットは返金の対象になりません。
  3. サブスクリプション（運営プラン／クラウドバックアップ）：自動更新されます。解約は各ストアの設定（iOS：設定 > Apple ID > サブスクリプション／Android：Google Play > お支払いと定期購入）から、更新日の24時間前までに行ってください。
  4. クラウドバックアップ（まもるパック）：解約後90日間はデータの復元のみ可能で、その後データは完全に削除されます。削除されたデータは復元できません。
  5. 返金は各ストアの返金ポリシーに従います。当社が個別に返金を行うことはできません。返金が行われた場合、対応するチケット・特典は取り消されます。

第4条（撮影コンテンツの権利と責任）

  1. 運営ユーザーは、撮影に際し、被写体となる方から撮影および本サービスでの利用（ファンユーザーへの提供を含む）についての同意を得る責任を負います。
  2. スナドロの著作権その他の権利は、撮影した運営ユーザーまたは正当な権利者に帰属します。当社はホスティング（受け渡しと保存）のために必要な範囲でのみコンテンツを取り扱います。
  3. ファンユーザーは、受け取ったスナドロを私的利用の範囲で楽しむことができます。被写体・撮影者の許可なく、商用利用・改変・第三者への再配布・SNS等での公開を行うことはできません（運営ユーザーが別途許可した場合を除きます）。
  4. サーバー上のスナドロは撮影から5日後に自動削除されます。期限内に受け取られなかったスナドロについて、当社は責任を負いません。

第5条（トレカ・ガチャ・交換）

  1. トレカはスナドロ受け取り時の抽選（ガチャ）で提供されます。提供割合はアプリ内に表示します。
  2. トレカ・チケット等のデジタルアイテムは金銭的価値を持たず、換金・払い戻しはできません。
  3. トレカの交換はアプリ内の交換機能によってのみ行えます。アプリ外での売買・現金や物品との交換（リアルマネートレード）は禁止します。
  4. 交換は両者の確認のうえ成立し、成立後の取り消しはできません。

第6条（禁止事項）

利用者は、以下の行為をしてはなりません。

  • 法令または公序良俗に違反する行為
  • 被写体の同意なく撮影・配布する行為、他者の肖像権・プライバシー・著作権等を侵害する行為
  • スナドロ・トレカをアプリ外で売買する行為（リアルマネートレード）
  • 不正アクセス、リバースエンジニアリング、チート行為、抽選結果や残高の改ざんを試みる行為
  • QRコードの不正な複製・転売・第三者への譲渡による不正取得
  • 他の利用者への嫌がらせ、なりすまし、迷惑行為
  • 本サービスの運営を妨害する行為

第7条（サービスの変更・停止）

当社は、メンテナンス・障害・その他やむを得ない事情により、本サービスの全部または一部を予告なく変更・中断・終了することがあります。サービス終了の際は、合理的な期間をもって事前に告知します。

第8条（免責）

  1. 当社は、本サービスに事実上または法律上の瑕疵がないことを保証しません。
  2. 通信環境・端末の状態等によりスナドロの受け渡しが完了しなかった場合、当社は再提供に努めますが、データの完全性を保証するものではありません。
  3. 運営ユーザーとファンユーザーの間、または利用者と第三者との間で生じた紛争（肖像権・チェキ会等のイベント運営に関するものを含む）について、当社は当事者とならず、責任を負いません。
  4. 当社が損害賠償責任を負う場合、その額は、当該利用者が直近6か月間に当社に支払った金額を上限とします（当社の故意または重過失による場合を除く）。

第9条（規約の変更）

当社は、必要に応じて本規約を変更できます。重要な変更がある場合は、アプリ内または本ページで事前に告知します。変更後に本サービスを利用した場合、変更後の規約に同意したものとみなされます。

第10条（準拠法・管轄）

本規約は日本法に準拠します。本サービスに関して紛争が生じた場合、東京地方裁判所を第一審の専属的合意管轄裁判所とします。
